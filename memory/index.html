Provider profiles
The fast version

What you actually need to know about each one.

🧠
Mem0
Apache 2.0 · mem0ai/mem0

LLM-powered memory layer. Extracts, stores, and retrieves structured facts across sessions. The most widely integrated option — first-class SDKs for Python, TypeScript, and OpenAI-compatible APIs.

Freemium Cloud + self-host Python SDK
Best for: agents that need reliable fact recall across long conversations, teams that want a managed service without vendor lock-in.
Benchmark: LongMemEval-S 67.6%
⚡
Supermemory
Proprietary · cloud API

Optimized for search-heavy workloads. Indexes content from many sources and surfaces it via semantic search. Generous free tier (1M tokens). Core engine is closed source.

Generous free tier Cloud only Search-first
Best for: knowledge wikis, chatbots that need to surface relevant content from a large corpus.
Benchmark: LongMemEval 81.6% (with GPT-4o)
🤖
ByteRover
CLI: MIT · cloud: proprietary

Highest benchmark score on temporal and multi-hop reasoning. CLI is open source and runs locally for free. Cloud sync costs $19/mo. Built for coding agents that need to track context across many sessions.

Local CLI free Local + cloud Multi-hop recall
Best for: coding agents, autonomous workflows, anything requiring multi-hop or temporal reasoning.
Benchmark: LoCoMo 92.19% (single-hop 95.4%, temporal 94.4%)
🔒
Hindsight
MIT · local + optional cloud

Three-stage memory: retain (ingest), recall (retrieve), reflect (synthesize). Full features run entirely locally for free. Cloud is pay-per-token if you need sync across devices.

Local = free Local + cloud Retain / Recall / Reflect
Best for: privacy-conscious setups, air-gapped agents, local-first development.
✨
Holographic
MIT · cloud

Completely free cloud service. MIT-licensed. Simplest possible on-ramp — no billing, no usage caps announced. Trade-off: no self-host option, limited documentation.

100% free Cloud Low overhead
Best for: experiments, hobby projects, and anyone who needs memory working in minutes with zero cost.
🏠
OpenViking
AGPL-3.0 · self-hosted

Fully open, self-hosted only. Free to run, zero external calls. AGPL license means modifications to the server must be disclosed if distributed — fine for internal use, check before building a SaaS product on it.

Free forever Self-host only AGPL-3.0
Best for: on-prem deployments, regulated industries, teams who need full data sovereignty.
👤
Honcho
AGPL-3.0 · cloud

Focuses on user modeling — building a structured representation of who a user is, not just what they said. "Dialect" concept distills preferences and behavior into persistent user profiles. $100 free credits to start.

$100 free credits Cloud User modeling
Best for: product agents that need to personalize responses, multi-user apps where each user's context matters.
🗃️
RetainDB
Proprietary · cloud only

Database-style memory with structured schema. No free tier — starts at $20/mo. Good if you need explicit control over what gets stored and how it's queried. Enterprise plans available.

No free tier Cloud only Structured schema
Best for: production apps that need structured data recall with predictable query behavior.
Use case recommendations
Just tell me what to pick

Top choices for each situation — ranked by fit, not by sponsorship.

💻
Coding agents

Tracking state across long sessions, referencing prior decisions, multi-hop context.

  • 1
    ByteRover — best benchmark score for multi-hop and temporal recall; local CLI is free
  • 2
    Mem0 — mature SDK, works well with tool-calling patterns, self-hostable
📚
Knowledge wiki / RAG

Indexing and retrieving from a large, growing body of content.

  • 1
    Supermemory — highest recall benchmark, built for search-heavy workloads
  • 2
    Mem0 — solid recall + open license if you need to self-host the index
⚖️
All-around / default pick

You want something reliable that handles most cases without overhead.

  • 1
    Mem0 — best combination of free tier, SDK quality, benchmarks, and self-host option
  • 2
    Hindsight — full features free locally, MIT, transparent architecture
🆓
Free / open source priority

You want zero cost and/or full access to the source code.

  • 1
    Holographic — MIT, everything free, no limits announced — fastest start
  • 2
    Hindsight — MIT, local-first, full features without a cloud account
  • 3
    OpenViking — AGPL, self-hosted, no external calls at all
🔐
Privacy / on-prem

Data cannot leave your infrastructure. No cloud calls.

  • 1
    OpenViking — self-host only, no external dependencies, AGPL
  • 2
    Hindsight — full local mode, MIT, no cloud required
  • 3
    Mem0 — self-host option with on-prem enterprise plan for full control
🏢
Enterprise / production

SSO, audit logs, SLA, on-prem support, no open-source copyleft risk.

  • 1
    Mem0 — Apache 2.0, on-prem tier, SSO, audit — covers the checklist
  • 2
    RetainDB — structured schema, enterprise plan, proprietary (no copyleft)
